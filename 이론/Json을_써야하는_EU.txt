CSV : 콤마와 엔터로만 데이터를 구분하는 파일 포맷
장점 : 저용량 데이터
단점 : 나쁜 가독성, 파이썬 전용

XML : 태그, 속성을 이용해 데이터를 저장하는 포맷

    <person>
    <이름>안현진</이름>
    </person>


JSON : javascript 객체 형식으로 데이터를 저장하는 포맷
전달할 때 문자열로 전달
데이터들이 key : value 형태로 짝지어져있다.
장점 : 가독성 좋음, 만들기 쉬움, 호환성이 좋음

    [	
        {
            "name" : "채정배",
            "age" : "20",
            "tel" : "01066152579"
        },
    ]
