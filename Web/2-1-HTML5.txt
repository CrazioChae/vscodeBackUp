1. HTML5 개요
 1. HTML 페이지 기본
  - HTML5 페이지의 기본 구조
   <!DOCTYPE html>
   <!--주석-->
   <html>
    <head>
        문서 제목, JS code, CSS code, metadata definition
    </head>
    <body>
        문서의 본문 텍스트, 이미지, 테이블, JS code, 동영상 등
    </body>
   </html>
  
  - HTML5 페이지의 필수 태그
   <!DOCTYPE html> - HTML5 문서임을 알리는 지시어
   <html>, <head>, <title>, <body> 태그


 2. HTML 태그의 특징
  - html 태그 구성
   태그이름 - 속성 (속성 이름, 속성 값)
  
  - 시작태그와 종료태그가 모두 있는 경우 : <html></html>

  - 시작 태그만 있는 경우 : <br>

  - 태그와 속성은 대소문자 구분 없음

  - 속성 값에 불필요한 공백 문자, HTML5 표준에 어긋남



2. 기본 문서 만들기
 1. HTML 기본 문서 만들기
  - 웹 페이지 타이틀 달기
  
  - <h1>~<h6> 태그로 문단 제목 달기
  
  - <p>로 단락 나누기

  - <hr> 태그로 수평선 긋기

  - <br> 태그로 새 줄로 넘어가기


 2. 문자, 기호, 심볼 입력
  - HTML5의 문자 : 유니코드 문자셋, UTF-8 코드 체계

  - 예약어, 키보드로 입력이 어려운 기호들, 심볼
   &엔터티; 혹은 &#코드값;
   ex) < --> &lt; 혹은 &#60;

  - 특수 문자, 기호, 심볼 삽입
   &divide, &radio, $nbsp;

  - <pre> 태그로 개발자의 포맷 그대로 출력
   공백 등의 포맷을 전부 출력

  - 텍스트 꾸미기
   <b><em><strong> 등


 3. 블록 태그와 인라인 태그
  - 태그 : 블록 태그와 인라인 태그로 구분
   블록 태그 사례 : <p> <h1> <div> <ul>
   인라인 태그 사례 : <strong> <a> <img> <span>

  - 블록 태그 
   1) 항상 새 라인에서 시작하여 출력
   2) 양 옆 다른 콘텐트를 배치하지 않고 한 라인 독점 사용
   3) 가장 많이 사용되는 블록 태그 : <div>

  - 인라인 태그
   1) 블록 속에 삽입되어 블록의 일부로 출력
   2) 가장 많이 사용된 인라인 태그 : <span>


 4. 메타 데이터 삽입
  - 메타 데이터
   데이터를 설명하는 데이터

  - HTML 페이지에 대한 메타 데이터를 담기 위한 태그들
   base, link, script, style, title, meta

  - 메타 태그들은 <head> 태그 안에 작성
   script는 body 내에도 작성 가능

  - base 태그
   웹 페이지들의 기본 URL과 페이지가 출력될 윈도우 지정

  - link 태그와 meta 태그
   link 태그는 외부 자원 연결에 사용
   
   meta 태그는 다양한 메타 데이터 표현
     웹 페이지의 저작자, 문자 인코딩 방식, 내용
     웹 페이지의 내용 설명
     웹 페이지의 키워드(검색 엔진을 위한)

    

3. 고급 문서 만들기
 1. 이미지 삽입
  - img 태그의 src 속성에 이미지 파일의 주소 지정
   src에 지정할 수 있는 이미지 종류
     ㄴBMP, GIF, PNG, JPG(JPEG), animated-GIF
  
  - img 태그로 이미지 삽입
   현재 이미지 파일의 경로명은 상대 경로명을 권장
   사진이 없는 경우 alt에 지정된 텍스트 출력


 2. 리스트 만들기
  - 3가지 종류의 리스트
   순서 있는 리스트 > ordered list - ol
   순서 없는 리스트 > unordered list - ul
   정의 리스트 > definition list - dl

  - 리스트 아이템 : li(생략 가능)


 3. 표 만들기, table
  - 표 만드는데 사용되는 태그들
   table : 표 전체를 담는 컨테이너
   caption : 표 제목
   thead : 헤딩 셀 그룹
   tfoot : 바닥 셀 그룹
   tbody : 데이터 셀 그룹
   tr : 행(td, th 포함)
   th : 열 제목 셀
   td : 데이터 셀

  - 행과 열 만들기
   표는 여러 행으로 구성
