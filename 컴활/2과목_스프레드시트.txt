


    01. 스프레드시트


    스프레드시트(Spreadsheet)
개념: 표 형식으로 데이터의 조직, 분석, 계산, 처리, 저장을
 하는 상호작용 컴퓨터 프로그램


    엑셀의 기본기능
파일 메뉴 > 옵션 단추
 일반 탭
  미니도구모음표시
  실시간 미리보기
  글꼴과 크기, 포함할 시트수

 고급 탭
  Enter키 누른 후 다음 셀, 행 및 열 머리글
  소수점 자동 삽입, 채우기 핸들, 눈금선 표시
  셀 내용 자동완성/직접편집, 수실을 셀에 표시
  사용자 지정 목록(내가 원하는 데이터목록)

 리본 사용자 지정
  리본 사용자 지정 > 개발도구 체크

 언어 교정
  영어 입력시 한글로 바뀔 때 > 한영 자동고침 체크해제

 빠른실행도구모음
  명령을 추가하면 일련번호로 된 바로가기가 지정됨


    통합문서
통합문서 메뉴
 새 파일작성: Ctrl + N
 파일 열기: Ctrl + O
 파일 닫기: Ctrl + F4, Ctrl + W

통합문서 공유(다른 사람과 같이 사용)
 검토 탭
 입력, 편집 가능
 조건부서식, 차트, 시나리오, 데이터 표, 피벗테이블 등은 불가능
 *상위버전 작성 문서는 하위버전에서 사용 불가능


    저장기능
암호지정
 대소문자를 구분, 문자, 숫자, 기호
 다른이름으로 저장 > 도구 > 일반 옵션
  열기 암호 255자까지
  쓰기 암호 15자까지

엑셀문서 저장 형식
 .XLSX 통합문서 파일
 .XLSM 매크로 포함
 .XLTX 서식파일
 .XLTM *매크로 서식파일
 .XLK 백업파일
 .XLW 작업영역 저장파일


    포인터 이동
Shift + Tab, Tab: 좌우로 이동
Shift + Enter, Enter: 상하로 이동
Alt + Page Up, Page Down: 한 화면 좌우로 이동
Ctrl + Page Up, Page Down: 현재 시트의 앞뒤 시트로 이동
Ctrl + 방향키: 데이터 범위의 상하좌우 끝으로 이동
Home: 행의 처음(A열)으로 이동
Ctrl + Home: 워크시트의 A1셀로 이동
Ctrl + End: 데이터 범위의 맨 오른쪽 아래의 셀로 이동
Page Up, Page Down: 한 화면 상하로 이동
F5: 셀 주소를 직접 입력해 이동


    셀 범위 지정
연속된 셀
 마우스로 드래그
 Shift 누른채로 마지막 셀 클릭
 Shift 누르고 방향키

떨어진 셀
 Ctrl 누른채로 셀 클릭

행과 열
 행 전체: Shift + SpaceBar
 열 전체: Ctrl + SpaceBar

워크시트 전체
 Ctrl + A, Ctrl + Shift + SpaceBar


    메모 삽입
메모 삽입: 검토 탭(Shift + F2)
오른쪽 상단에 빨간색 삼각형 점
데이터를 지워도 메모는 그대로
셀 이동하면 메모도 이동


    윗주(내천)
데이터 위쪽에 표시, 문자 데이터에만 삽입
윗주 삽입, 수정: 윗주편집
윗주 글꼴, 맞춤: 윗주설정
윗주서식은 윗주 전체에 대해서만 적용


    행, 열 삽입/삭제
삽입 Ctrl + +
삭제 Ctrl + -


    행, 열 크기변경
행 높이 변경
 해당 행의 글꼴 크기 중 가장 큰 것에 맞추어 자동으로 조절
 여러 개의 행을 선택한 후 높이를 조절 > 모두 동일하게 조절
 서식 > 행 높이

열 너비 변경
 여러 개의 열을 선택, 동일한 너비로 조절
 서식 > 열 너비


    이름정의(이름상자 클릭하고 엔터키), Ctrl + F3
절대참조
반드시 문자로 시작, 공백 미포함
최대 255자 가능


    워크시트 기능
워크시트 선택
 연속적인 시트: Shift
 떨어져 있는 시트: Ctrl
 모든 시트: 오른쪽 버튼 > 모든 시트 선택

워크시트 삽입(Shift + F11)
 하나의 통합 문서에는 기본적으로 3개의 워크시트 포함
 최대 255개까지 추가
 시트 삽입하면 선택한 시트의 오른쪽에 삽입
 여러 개 시트를 선택 후 시트를 삽입하면 선택한 수만큼의 시트가 삽입
 떨어져 있으면 삽입 불가

워크시트 삭제
 *삭제된 시트는 되살릴 수 없다
 여러개 시트를 선택해 한번에 삭제 가능

워크시트 이동/복사
 시트를 복사할 때마다 시트 이름에 ()가 삽입되어 (2), (3)으로 일련번호 붙음
 이동할 시트 선택 후 드래그하면 이동
 복사할 시트 선택 후 Ctrl 누른채 드래그하면 복사



    02. 스프레드시트 일반


    문자데이터
셀의 왼쪽으로 정렬
숫자 데이터 앞에 접두어(')를 입력하면 문자로 인식
한 셀에 2줄 입력: Alt + Enter
범위를 설정후 모두 입력: Ctrl + Enter


    숫자데이터
셀의 오른쪽으로 정렬
중간에 공백이나 특수 문자가 있으면 문자로 인식
음수 표현: 숫자 앞에 - 혹은 ()로 묶음
분수는 0을 입력, 한 칸 띄고 입력
셀의 너비보다 긴 경우 지수 형식(12345 > 1.2345*10000 = 1.2345E + 04)


    날짜데이터
셀의 오른쪽으로 정렬
하이픈(-)이나 슬래시(/)를 이용
시간 데이터: 콜론(:)을 이용, 12시간제 표시: AM이나 PM을 입력
연도가 두자리일때 30이상은 1900년대, 29이하면 2000년대
오늘날짜: Ctrl + ;
현재시간: Ctrl + Shift + ;


    채우기핸들
숫자데이터
 한 셀: 드래그 > 복사, Ctrl 누르고 드래그 > 1씩 증가
 두 셀: 첫 값과 둘째 값의 차이만큼 증가/감소

문자데이터
 동일한 데이터가 복사됨

혼합데이터
 한 셀: 오른쪽에 있는 숫자는 1씩 증가
 두 셀: 숫자 데이터는 차이만큼 증가/감소

날짜데이터
 한 셀: 1일 단위로 증가
 두 셀: 두 셀의 차이만큼 년월일 단위로 증가

사용자정의목록
 등록한 목록 순서대로 목록이 표시
 파일 > 옵션 > 고급 > 사용자 지정 목록 편집


    데이터수정
전체수정
 새로운 데이터 입력
 여러 데이터 동시 수정: 수정할 셀 선택하고, Ctrl + Enter
부분수정: 더블클릭 또는 F2
삭제: Del


    선택하여 붙여넣기
수식: 수식만 복사
값: 화면에 표시된 값만 복사
서식: 셀 서식만 복사
연산: 복사한 데이터와 붙여넣기할 위치에 있는 데이터를 연산자로 계산
행/열 바꿈: 행과 열의 위치를 서로 바꾼다.
연결하여 붙여넣기: 원본이 수정되면 붙여넣기한 셀도 자동 수정
