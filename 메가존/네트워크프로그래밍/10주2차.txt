  - Logical Byte Size

 + Data Type

  - Image Type

    (3)명령어 처리부와 데이터 전송부간의 동기화

 + 데이터 전송부에 의한 명령어 처리부의 반응 관리
  - File등의 전송 종료시, 명령어 처리부에 알람
