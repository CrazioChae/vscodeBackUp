    1. 조건문

    (1)조건문과 순서도
 + 조건문: 컴퓨터가 스스로 일을 하도록 하는 것
 + 순서도
  - 프로그램 시작
  - 작업
   이전 작업이 끝나고 시작
   해당 작업 종료 후 다음 작업으로 이어짐
  - 조건
   true와 false가 결과로 나오는 관계 연산 혹은 논리 연산이 수행
   true일 때와 false일 때 서로 다른 작업으로 이어짐
  - 프로그램 종료


    2. if 조건문

    (1)if-else
// 작업 1
if(판단){
    // true일 경우: 작업 3
}else{(생략 가능)
    // false일 경우: 작업 2
}
// 작업 4


    1. If 조건문
변수에 저장된 상황 정보를 이용하여 차후 수행할 작업이 달릴 수 있게 함
중첩이 가능함
디버깅을 통해 조건문의 정확성을 확인할 수 있음

    2. 디버거
프로그램의 수행 상황을 보여줌
Line 단위로 수행함
매 순간의 변수 값의 변화를 확인할 수 있음
조건문에 따른 진행 상황을 파악할 수 있음
