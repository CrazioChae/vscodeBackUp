    1. 제너릭 클래스와 함수 선언

    (1)제너릭 클래스와 인터페이스 선언
 + 클래스 이름 뒤에 <>에는 클래스를 매개 변수처럼 나열
  - 타입 매개 변수라 부르며, 개수 제한은 없음
 + 타입 매개 변수는 클래스(인터페이스 포함)여야만 함
  - 기본형을 필요로 한다면, 매칭된 클래스를 사용할 것
 + 타입 매개 변수를 static 멤버 변수에 적용하면 안됨
  - Generic Class 선언 /w static member variable
 + 서로 다른 타입 매개 변수를 가진 Generic Class 생성


    1. 제너릭 클래스/인터페이스 선언과 사용
타입 매개 변수를 사용
static 멤버 변수에는 적용 불가
클래스 선언에 사용하는 타입 매개 변수는 static 멤버 함수에 적용 불가
타입 매개 변수를 사용한 객체 생성과 배열 생성은 불가능(변수 선언은 가능)

    2. 제너릭 함수의 선언과 사용
클래스에 적용된 타입 매개 변수와 별도의 타입 매개 변수를 사용 가능
매개 변수와 반환 값의 타입에 적용 가능
