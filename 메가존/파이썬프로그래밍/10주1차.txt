    1. 함수의 기본

    (1)함수란?
 + 파이썬에서 프로그램을 쪼개는 3가지 방법
  - 함수(function): 여러 개의 명령을 하나의 단위로 묶어 놓은 것
  - 클래스(class): 코드 중, 관련된 변수와 함수르 하나로 묶은 것
  - 모듈(module): 함수를 모아놓은 프로그램, 독립적인 파일
 + 특정 기능을 구현하는 부분을 따로 떼어 구현
 + 반복적으로 사용되는 의미 있는 부분
 + 서브루틴, 루틴, 메서드, 프로시저
 + 함수는 특정 작업을 수행할 목적으로,
  입력 받으면 연산하고 결과를 출력함
 + 함수의 두 가지 유형
  - 내장 함수: 파이썬이 제공
  - 사용자가 작성하는 함수
 + 함수를 호출
  - 함수 사용
 + 매개변수
  - 함수 호출 시 괄호 내부에 넣는 여러 자료
 + 리턴값
  - 함수를 호출해 최정적으로 나오는 결과
 + 함수 = 코드의 집합

    (2)함수의 개념과 필요성
 + 함수(function): 입력 > 연산 > 출력
 + 메서드와의 차이점: 메서드는 클래스 내부

    (3)함수를 작성하고 호출하기
 + 함수 작성 => 함수 정의
 + 함수 사용 => 함수 호출
 + 프로그램을 일목요연하게 보기 편함
 + 함수 정의문의 형식
 + 함수는 def 키워드를 이용하여 정의
def function_name(a, ...):
    docstring

    2. 함수에 인수 전달하기

    (1)함수에 1개의 인수 전달하기
 + 함수를 사용할 때 필요 정보를 전달
 + 인수(argument): 함수에 실제 전달되는 값
 + 매개변수(parameter): 인수를 담아 함수 내에서 쓰이는 변수

    (2)함수에 여러 개의 인수 전달하기

    3. 함수의 값 반환하기

    (1)반환 값(return value)이란?
 + 함수로부터 되돌아오는 결과(출력값)
 + 파이썬에서는 함수가 여러 값을 반환 가능


    1. 함수
특정 기능을 구현하는 부분을 따로 떼어 구현한 것
반복적으로 사용되는 의미 있는 부분
내장 함수와 사용자가 작성하는 함수 2가지로 분류
프로그램을 함수화하면 프로그램의 흐름을 일목요연하게 볼 수 있음

    2. 함수에 인수 전달하기
사용자는 함수를 사용할 때 작업에 필요한 정보(값)를 전달할 수 있음
인수(argument): 호출 프로그램에 의하여 함수에 실제로 전달되는 값
매개변수(parameter): 인수를 담아 함수 내에서 쓰이는 변수

    3. 함수의 값 반환하기
반환 값(return value): 함수로부터 되돌아오는 값
파이썬에서는 반환 값을 사용하지 않거나 하나의 함수가
 여러 개의 값을 반환하도록 할 수 있음
