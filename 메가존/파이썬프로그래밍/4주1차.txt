    1. 기본 if문

    (1)조건식
 + 식을 또 다른 식과 관계 연산자를 사용해 비교하는 식
 + 문자열 비교
  - 문자열 비교 = 유니코드 비교
 + 논리 연산자
  - and , or, not

    (2)if문의 종류
 + if: 단일 대안 선택
 + if-else: 양자 택일 선택
 + 중첩 if: 다자 택일 선택
 + if-elif-else: 다자 택일 선택

    (3)if
 + 파이썬은 들여쓰기가 매우 중요
 *+ if문 다음에 실행할 문장은 들여쓰기를 해서 작성
 + 들여쓰기 할 때는 tab보단 spacebar 4번 입력하여 사용하는걸 권장

    (4)if~else
 + 조건이 참일 때와 거짓일 때 실행할 문장이 다름


    2. 복합 if문

    (1)if~else~if~else
 + if문 내부에 if문 사용

    (2)if~elif~else


    3. if문 응용

    (1)리스트와 함께 사용
 + 리스트(List)
  - 데이터 여러 개를 한곳에 담아 놓은 것
  - 방법: 대괄호[]로 묶고 그 안에 넣음

    1. 조건식
식(또는 변수, 값)을 또 다른 식(또는 변수, 값)과
 관계 연산자(비교 연산자)를 사용하여 비교하는 식
두 개의 문자열을 비교하는 것은 문자열에 해당하는 유니코드를 비교하는 것
논리 연산자(and, or, not)는 두 개의 단순한 조건식을 연결하여
 복잡한 조건식을 만듦

    2. if 문의 종류
if 문: 단일 대안 선택 구조
if~else 문: 양자 택일 선택 구조
중첩된 if 문: 다자 택일 선택 구조(이것보다 if~elif~else 문이 좋음)
if~elif~else 문: 다자 택일 선택 구조
