    1. 모듈의 개요

    (1)모듈이란
 + 하나의 파일에 함께 저장된 함수
 + 연관된 함수들을 모듈 안에 함께 모아둠
 + 외장 모듈에 포함된 함수를 호출하려면 프로그램 상단에서
  import 키워드를 사용해 모듈을 명시
 + 모듈: 함수의 집합

    (3)내장 함수와 모듈의 차이
 + 내장 함수
  - 어떤 모듈을 import하지 않고 언제든지 호출하여 사용하는 함수

    (4)파이썬 표준 라이브러리
 + math: 수학 연산
 + os: 파일, 프로세스 연산
 + string: 문자열 연산
 + time: 날짜, 시간 연산


    2. 모듈을 import하는 방법

    (1)import하는 형식
 + 필요한 모듈을 import 하는 세 가지 형식
  - import 모듈
  - from 모듈 import 함수1, 2...
  - from 모듈 import *
 + import 모듈 as 별칭

    (5)사용자 정의 모듈
 + 스스로 작성한 함수들을 모아 모듈에 저장
 + 표준 라이브러리 모듈과 같은 방법으로 import하여 사용
 + 모듈 저장 파일의 확장자는 .py


    1. 모듈이란?
함수의 집합, 즉 하나의 파일에 함께 저장된 기능적으로
 연관된 함수(메소드)들
내장 함수가 아닌, 어떤 모듈에 포함된 함수를 호출하려면
 프로그램의 상단에서 import 키워드를 사용해서 그 모듈을 명시해야 함
내장 함수는 어떤 모듈을 import하지 않고도,
 언제든지 호출하여 사용할 수 있는 함수

    2. 사용자 정의 모듈
사용자가 스스로 작성한 함수들을 모아서 모듈에 저장한 것
파이썬의 표준 라이브러리에 속한 모듈과 같은 방법으로 impor하여 사용
사용자 정의 모듈의 이름은 파이썬의 표준 라이브러리에 포함된 모듈의 이름과 같지 않아야 함

    3. dir( )과 help( ) 함수
dir( ) 함수: 모듈 안에 어떤 함수들이 포함되어 있는지 알려줌
help( ) 함수: 함수에 관한 도움말(독스트링)을 보여줌
