    1. 반복문 이해

    (1)반복문이란
 + 사용하는 이유: 반복 작업에서 효율적인 코드 작성을 위함
 + 루프(loop)라고 부름
 + 반복문의 몸체 = 반복해서 수행하는 문장들

    (2)반복문의 종류
 + 파이썬의 두 가지 반복문
  - for문: 정해진 횟수만큼 반복
  - while문: 반복 횟수를 미리 알 수 없는 경우


    2. for문 이해

    (1)for문
 + 가장 흔히 사용하는 반복문
 + 형태
for 변수 in 시퀀스:
    문장 블록을 네 칸만큼 들여쓰기(1탭)
for 변수 in range(정수):
    "
    (3)for문의 필요성
 + 원하는 횟수, 범위만큼 반복
 + 리스트, 문자열 등 열거 타입도 순차적으로 출력함


    3. 중첩 for문

    (1)중첩 반복문
 + 반복 루프 내 반복 루프를 만듦

    (2)중첩 반복문의 예
for y in range(5):
    for x in range(10):
        print('*', end = "")
    print("")

for y in range(1, 6):
    for x in range(y):
        print("*", end="")
    print("")


    1. 반복문
많은 한 개 이상의 문장들의 블록을 어떤 조건이 만족되는 동안 반복해서 수행
흔히 루프(loop)라고 부름
반복문의 몸체: 반복해서 수행하는 문장(들)

    2. 파이썬의 두 가지 반복문
　1) for 문
　　- 정해진 횟수만큼 반복
　　- for 변수 in 시퀀스: 흔히 시퀀스로 문자열 또는 리스트를 사용
　　- for 변수 in range(정수): 흔히 정해진 횟수만큼 반복할 때 사용
　　- 중첩 for 문: 프로그램에서도 반복 루프 안에 다시 반복 루프가 있을 수 있음
　2) while 문
　　- 반복 횟수가 정해진 경우
　　- 또는 반복 횟수를 미리 알 수 없는 경우에도 사용
