    1. 자료형의 이해

    (1)자료(data)
 + 프로그램이 처리할 수 있는 모든 것

    (2)자료형(data type)
 + 자료를 기능과 역할에 따라 구분한 것
  - 문자열 String
  - 숫자 Number
  - 논리 Boolean

    (3)자료를 알아야 하는 이유
 + 자료를 자료형에 맞게 모으고 처리해
  더 큰 자료형을 만들어 나가 프로그램을 완성
 + 가장 기본 단위로서의 자료의 의미와 쓰임새를 아는 것이 중요

    (4)기본 자료형
 + 자료의 형식
 + type() 함수로 확인
print(type("Hello"))
 >> <class 'str'>


    2. 자료형 분류

    (1)파이썬에서의 자료의 종류
 + 모든 프로그램은 자료를 처리
 + 프로그래밍에서 가장 기본적인 자료형: 정수, 실수, 문자열

    (2)정수형과 실수형
 + 수치형 데이터는 산술 연산 가능
 + 정수형(int)
x = 123
print(x)
type(x)
 >> <class 'int'>
 + 실수형(float)


    3. 배정문

    (1)배정문
 + 변수에 값을 배정하는 문장
number / = / 1234
변수 / 배정연산자 / 식
 + 변수를 생성하기 전에 참조(사용)
 + 어떤 변수도 가리키지 않는 주기억장치 공간을 회수
 
    (2)쓰레기 수집
 + 새 값이 할당 될 때
i = 99
i = 12.3
 + 변수 i를 식에 사용할 때
i = i + 1

    (4)두 변수의 값을 교환
 + 임시 변수를 사용해 A, B의 내용을 교환


    1. 자료(data)
프로그램이 처리할 수 있는 모든 것
프로그램은 자료를 처리하기 위한 모든 행위

    2. 자료형(data type)
자료를 기능과 역할에 따라 구분한 것
문자열(string): 메일 제목, 메시지 내용 등
숫자(number): 물건의 가격, 학생의 성적 등
불(boolean): 친구의 로그인 상태 등
