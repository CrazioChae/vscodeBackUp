    1. 문자열의 이해

    (1)문자열(String)
 + 따옴표로 둘러싸 입력하는, 글자의 나열
 + 큰따옴표로 문자열 만들기
print("안녕하세요.")
 + 작은따옴표로 문자열 만들기
print('안녕하세요.')

    (2)이스케이프 문자(Escape Character)
 + 역슬래시 기호와 함께 사용하는 특수 문자
\" => 큰따옴표 출력
\' => 작은따옴표 출력
\n => 줄바꿈
\t => tab 의미

    (3)리스트 코드와 비교
 + 리스트는 대괄호 []로 묶고 문자열은 작은따옴표로 묶어 출력
 + 더하기(+) 기호 사용해 문자열 연결
 + 곱하기(*) 기호 사용해 문자열 반복


    2. 문자열 연산자

    (1)문자열 연결 연산자: +
 + 숫자에는 사칙연산 연산자
 + 문자에는 문자열 연결 연산자
 + 문자열과 숫자 사이에는 사용 불가
 + 문자끼리 숫자끼리 연산

    (2)문자열 반복 연산자: *
 + 문자열을 숫자와 *연산자로 연결

    (3)문자 선택 연산자(인덱싱): []
 + 문자열 내부의 문자 하나를 선택
 + 대괄호 안에 선택할 문자의 위치를 지정
 + 인덱스
  - 제로 인덱스: 0부터 시작
 + 문자를 거꾸로 출력하려는 경우, 대괄호 인덱스를 음수로 표현
  - [-1], [-2], [-3]

    (4)문자열 범위 선택 연산자(슬라이싱): [:]
 + 문자열의 특정 범위를 선택
 + 대괄호 안 범위를 콜론으로 표시


    3. 문자열 함수

    (1)format()함수로 숫자를 문자열로 변환
 + 중괄호 포함 문자열 뒤에 .format() 함수 사용
 + 중괄호 개수와 format함수 안 매개변수 개수는 같아야 함


    1. 이스케이프 문자(escape character)
역 슬래시 기호와 함께 조합해서 사용하는 특수한 문자
\“ : 큰따옴표를 의미
\‘ : 작은따옴표를 의미
\n : 줄바꿈 의미
\t : 탭 의미

    2. 문자열(string)
문자열 : 따옴표로 둘러싸 입력하는, 글자가 나열된 것

    3. 문자열 연산자
문자열 연결 연산자: +
문자열 반복 연산자: *
문자 선택 연산자(인덱싱): [ ]
문자열 범위 선택 연산자(슬라이싱): [:]

    4. 문자열 함수 format( )
숫자를 문자열로 변환
중괄호 개수와 format 함수 안 매개변수의 개수는 반드시 같아야 함
