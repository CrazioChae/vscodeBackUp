    1. 지역 변수

    (1)변수의 범위
 + 그 변수에 접근할 수 있는 프로그램의 일부분

    (2)지역 변수와 전역 변수의 이해
 + 지역 변수: 한정된 지역에서만 사용
 + 전역 변수: 프로그램 전체에서 사용

    (3)지역 변수
 + 변수를 만든 함수 안에서만 접근 가능
 + 함수 바깥에선 접근 불가능
 + 지역 변수에 접근할 수 있는 범위는 지역 범위(Local Scope)라고 함

    (4)nonlocal 지역변수
 + nonlocal은 현재 함수의 지역 변수가 아니라는 뜻
 + 바깥 함수의 지역 변수를 사용


    2. 전역 변수

    (1)전역 변수
 + 어떤 함수에도 속하지 않는, 메인 프로그램에서 선언한 변수
 + 메인 프로그램 내 모든 함수에서 접근 가능한 변수

    (2)global 키워드
 + 함수 func 안에서 새로운 지역 변수 a를 생성하지 않고
  메인 프로그램 전역 변수 a의 값을 변경하려면 함수 func의 
  첫 줄에서 global 키워드를 사용해 a가 전역 변수임을 명시

    (3)전역 상수
 + 함수를 수정할 수 없는 값
 + 관례상 전역 상수를 위한 이름은 모두 대문자로 표기


    1. 전역 변수
메인 프로그램에서 값을 배정한 변수
함수를 포함하여 스크립트 전체에서 접근 가능
전역 변수에 접근할 수 있는 범위는 전역 범위(global scope)
변수명 앞에 global을 붙여 선언

    2. 지역 변수
어떤 함수 안에서 값을 배정한 변수
매개변수 포함
한정된 지역에서만 사용되며, 함수가 실행되는 동안에만 존재
지역 변수에 접근할 수 있는 범위는 지역 범위(local scope)

    3. nonlocal 지역 변수
nonlocal은 현재 함수의 지역 변수가 아니라는 뜻이며
 바깥쪽 함수의 지역 변수를 사용함
