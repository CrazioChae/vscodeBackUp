    1. 리스트의 이해

    (1)리스트의 이해
 + 리스트는 변수를 한 줄로 붙인 뒤 전체의 이름을 지정
 + 각각은 aa[0], aa[2]...와 같이 번호를 붙여 사용
 + 리스트의 특징
  - 포함된 값을 원소 | 항목 이라 부름
  - 원소에는 순서가 있음 > 시퀀스
  - 여러 데이터 타입의 값들이 포함됨
  - 중복값이 허용
  - 가변 데이터 타입

    (2)리스트를 사용하는 이유
 + 일일히 데이터를 생성하여 계산하기 번거로움

    (3)리스트 생성 방법
 + 다른 언어의 배열에 해당하는 자료구조
 + 리스트는 동적: 필요에 따라 크기 조절이 가능
리스트_이름 = [값1, 값2...]
  - 쉼표로 구분
 + 리스트를 사용하면 인덱스로 선언
 + 빈 리스트 생성
a_list = []
zeroes = [0]*10
 + 여러 데이터 타입으로 이루어진 리스트
b_list = ["a", 9, 3.14]


    2. 리스트의 구조

    (1)리스트의 내부 구조
 + 주소를 가리킴

    (2)리스트와 인덱스
 + 양수 인덱스와 음수 인덱스
  - 앞에서부턴 0, 1, 2...뒤에서부턴 -1, -2, -3...

    (3)리스트의 색인
 + 리스트 타입 데이터에는 자동으로 색인이 생김
 + 색인 > 시퀀스
 + 0부터 정수 형태로 1씩 증가
 + len()함수를 사용해 리스트 길이 파악
len(b_list) =>> 3
 + 리스트 색인 오류
  - 호출한 색인이 길이를 초과하는 경우 > 오류


    1. 리스트 기본 실습

    (1)리스트
 + 여러가지 자료를 저장할 수 있는 자료
 + 자료들을 모아 사용할 수 있음
 + 대괄호 내부에 자료들 넣어 선언

    (2)요소

    (3)인덱스

    1. 리스트(list)
여러 가지 자료를 저장할 수 있는 자료
자료들을 모아서 사용할 수 있게 해 줌
대괄호 내부에 자료들을 넣어 선언
리스트의 원소들에는 순서가 있음 → 시퀀스
리스트에 중복된 값들이 존재할 수 있음
가변 데이터 타입

    2. 요소(element)
리스트의 대괄호 내부에 넣는 자료
항목이라고도 부름

    3. 인덱스(index)
리스트 타입 데이터에는 자동으로 ‘색인(인덱스)’이 생김
색인은 값을 찾아갈 수 있도록 차례대로 매긴 번호와 같은 개념
파이썬에서 제공하는 기본 색인은 0부터 시작하여 정수 형태로 1씩 증가하며, 음수 색인도 존재함
